CSS REFERENCE

Text

PROPERTY DESCRIPTION
font-family specify typeface to be used
font-size specify size for the font
font-weight controls how bold/thin text appears
Font-weight Values
Bold makes font bold
Normal default value; makes font normal
Lighter makes one font weight lighter
Bolder makes one font weight bolder
font-style specify typeface to be used
Font-style Values
Normal makes text appear in normal style
Italic text appears in italic
text-align specify alignment of text
Text-align Values
Left align to left side
Right align to right side
Center centers text
Justify takes up full width, aligns both left and right side
text-transform change the case of text
Text-transform Values
Uppercase text appear in uppercase
Lowercase text appear in lowercase

Colors

PROPERTY DESCRIPTION
color specify color of text
background-color style an elements background color
opacity measures how transparent an element is

Layout

PROPERTY DESCRIPTION
letter-spacing set horizontal spacing between individual character of an element
word-spacing sets space between words
line-height set vertical space between lines of text

Images

PROPERTY DESCRIPTION
background-image make background of an image
